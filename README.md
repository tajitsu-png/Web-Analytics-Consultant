# ウェブ解析士認定資格試験 演習問題集

## 概要

2026年ウェブ解析士認定試験公式テキスト（全8章）に完全準拠した4択演習問題集アプリです。
試験本番を想定した出題設計・詳細解説・進捗評価機能を備えています。

---

## 実装済み機能

### 🎯 演習機能
- 全8章を選択して演習スタート
- 1セッション10問（4択形式）
- 正しいものを選ぶ（タイプA）：5問 / 誤っているものを選ぶ（タイプB）：5問 の均等出題
- 前半ブロック3問・中盤ブロック4問・後半ブロック3問の物理分割ランダム出題

### 📝 回答・解説機能
- 選択肢クリックで即座に正誤判定
- 全選択肢（A〜D）の正誤理由を詳細解説
- 出典（章・小タイトル）を毎問表示

### 📊 進捗・評価機能
- ヘッダー進捗バー（リアルタイム更新）
- 10問完了後の正答数・正答率・評価コメント表示
  - 80%以上：十分理解
  - 60〜79%：要復習
  - 59%以下：再読推奨
- 全問の正誤一覧と不正解問題の正解表示

### 🔄 セッション管理
- 演習完了後は章選択に戻り、再演習・他章演習が可能
- 同一章の再演習で問題がランダム再抽出

## 問題設計ルール

- **出題根拠**：2026年ウェブ解析士認定試験公式テキスト限定
- **ブロック分割**：前半（0-33%）・中盤（34-66%）・後半（67-100%）に物理分割
- **出題配分**：前半3問・中盤4問・後半3問
- **タイプ配分**：タイプA（正しいものを選ぶ）5問・タイプB（誤っているものを選ぶ）5問
- **解説形式**：全選択肢（A〜D）の正誤理由を完全記載
- **出典表示**：章名・小タイトル名まで明記
